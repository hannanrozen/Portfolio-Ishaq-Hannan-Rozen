📖 Deskripsi

Website portfolio profesional yang menampilkan keahlian, proyek, dan informasi kontak sebagai Front-End Developer & UI/UX Designer. Website ini dibangun menggunakan HTML5, CSS3, dan JavaScript dengan desain yang responsive dan modern.

🛠️ Teknologi yang Digunakan

1. HTML5 - Struktur semantic dan modern
2. CSS3 - Styling dengan Flexbox, Grid, dan animasi
3. JavaScript (ES6+) - Interaktivitas dan DOM manipulation
4. Font Awesome - Icon library
5. Google Fonts - Typography (Poppins)

🎯 Section Website

1. Navigation - Menu navigasi dengan smooth scrolling
2. Hero Section - Intro dengan CTA buttons
3. About Section - Bio profesional dan skills
4. Projects Section - Showcase 3 project utama
5. Contact Section - Informasi kontak dan form
6. Footer - Copyright information
